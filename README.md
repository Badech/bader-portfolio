# Bader Echchalh - Portfolio Website

A premium, conversion-focused portfolio website for Bader Echchalh, specializing in web design and development for HVAC and local service businesses.

## 🎯 Project Overview

This website showcases Bader's expertise in creating high-converting websites for U.S. service businesses including HVAC companies, plumbers, electricians, and other local service providers. The site emphasizes business results, strategic positioning, and conversion optimization.

## 🛠️ Tech Stack

- **Framework:** React 18 with TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **UI Components:** Shadcn/UI (Radix UI primitives)
- **Animations:** Framer Motion
- **Routing:** React Router v6
- **Icons:** Lucide React
- **Form Handling:** React Hook Form + Zod

## 📁 Project Structure

```
src/
├── components/
│   ├── sections/        # Page sections (Hero, Services, etc.)
│   ├── ui/             # Reusable UI components (shadcn)
│   ├── Navbar.tsx
│   ├── Footer.tsx
│   └── Reveal.tsx      # Animation wrapper component
├── pages/              # Route pages
├── hooks/              # Custom React hooks
├── lib/                # Utility functions
└── assets/             # Images and static assets
```

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ and npm

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
